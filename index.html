Data Pipeline

Three free public APIs feed the dashboard — no API keys required. GitHub Actions fetches and snapshots data every 6 hours so the site renders instantly even before API calls resolve.

Questions

Frequently Asked Questions

What is Tale Waters & Tides?
Tale Waters & Tides is an Arkansas fishing intelligence platform that combines real-time USGS water data, NWS weather forecasts, and USACE lake levels to give anglers the most accurate picture of current fishing conditions. It's free, mobile-first, and requires no app installation.
Where does the water data come from?
Live data is sourced from three free public APIs: USGS Water Services (stream gauge temperatures, flow rates, gage heights), the National Weather Service (7-day forecasts), and the US Army Corps of Engineers (lake level and dam release data). No API keys are required. Data is also pre-fetched every 6 hours by GitHub Actions and stored as JSON snapshots so the site renders immediately.
Which Arkansas waters are covered?
The platform covers 39 Arkansas water bodies including Beaver Lake, Greers Ferry Lake, Bull Shoals Lake, Lake Maumelle, Lake Ouachita, DeGray Lake, the White River, the Buffalo National River, and 31 more lakes, rivers, tailwaters, and streams across the state.
What is the relationship with Pocket Fishing Guide?
Pocket Fishing Guide is the live app that powers Tale Waters & Tides data intelligence. This marketing site documents the platform, showcases features, and will serve as the home for the full product suite. The data pipeline, architecture, and best practices are borrowed directly from PFG.
How is the site deployed?
The site is deployed as a static GitHub Pages site with automatic deployment on every push to the main branch via GitHub Actions. See docs/DEPLOY-LIVE-SITE.md for the full deployment guide. No build step, no npm, no server required.
How can I contribute?
Local fishing knowledge is as valuable as code. File a bug report, request a feature, share local intel, or submit a PR on GitHub ↗. See the README for contribution guidelines.
