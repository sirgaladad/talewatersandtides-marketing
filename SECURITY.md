# Security Policy — Tale Waters & Tides Marketing Site

Thank you for helping keep **talewatersandtides.com** and this repository safe.

## Supported Versions
This project is maintained on the default branch:

- **Supported:** `main`
- **Not supported:** older forks / copies / other branches

## Reporting a Vulnerability (Preferred)
Please report security issues using **GitHub Private Vulnerability Reporting**:

1. Open the repo **Security** tab
2. Click **Report a vulnerability**
3. Include details requested below

## Alternate Private Reporting (Email)
If you cannot use GitHub Private Vulnerability Reporting, email:

**security@talewatersandtides.com**

Please include the repository name (**sirgaladad/talewatersandtides-marketing**) in the subject line.

**Do not post vulnerability details publicly** (issues, Discussions, social media, etc.).

## What to Include
- Description of the issue and user impact
- Steps to reproduce
- URLs affected (pages, assets), browser/device details if relevant
- Any error output (please redact secrets)
- Whether you think it could affect site visitors vs only repo maintainers

## Response Timeline (Best Effort)
- **Initial response:** within **7 days**
- **Fix / mitigation target:** within **30 days** (depending on complexity)

## Disclosure Policy
Please keep details private until we’ve deployed a fix or agreed on disclosure timing.

Thank you.
